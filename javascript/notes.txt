Variables: 
----------
    ->variables named memory location used to store data or values and it can change N number of times during execution 

        str="javascript"
        str=10

    1. variable Declaration 
    2. variable initialization 
    3. variable utilization 
    4. variable Re-initialization 
    5. variable Re-Declaration

In JS we can define variables in 3 ways : 
-----------------------------------------
    1. let 
        ->let variables acts like a local scop/ block scop 
        ->variable Declaration and initialization in same/ different line is possible 
        ->re-initialization is possible but re-Declaration is not possible 
        ->we can't hoist the let variables 

    2. var 
        ->var variables acts like a global scop/ block scop (because var present in windows object)
        ->variable Declaration and initialization in same/ different line is possible 
        ->re-initialization and re-Declaration is possible 
        ->we can hoist the var variables 
        
    3. const 
        ->const variables acts like a final (because once initialization values we can't modify data/values)
        ->variable Declaration and initialization in same is possible but different lines is not possible 
        ->re-initialization and re-Declaration is not possible 
        ->we can't hoist the const variables 






hoisting : 
----------
    ->utilizing the variables before Declaration 
    ->hoisting is possible only through 'var' variables
